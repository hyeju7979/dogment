    상단/측면/하단 디자인 개념

    모든 페이지에 상단, 측면, 하단 디자인을 하나하나 입력하는 불필요한 작업을 줄이실 수 있도록 나누어놓은 영역입니다.

    사용 설명

    상단, 측면, 하단의 각 영역 편집 후 생성되는 태그를 중앙 디자인 편집시 입력하시면 해당 영역의 디자인을 자동으로 가져옵니다.

    따라서 중앙 디자인 편집 시 상단/측면/하단 디자인 가상태그를 활용하여 자유로운 레이아웃 구현이 가능합니다.

    ※ 위 내용은 오른쪽 상단 메뉴얼을 클릭하시면 다시 확인하실 수 있습니다.

    메모 작성 폼

    사용자태그란?

    디자인 소스를 하나의 태그로 만들어 적용할 수 있는 기능으로 페이지 소스를
    대폭 간소화     시켜 줄 수 있고, 페이지 디자인을 영역별로 구분하여 편집할 경우
    편리하게 편집/수정하실 수 있습니다.

    또한 생성된 사용자태그는 모든 페이지에 적용하실 수 있기 때문에 모든 페이지에
    공통 노출되는 이벤트 배너 등의 소스를 사용자 태그로 생성 하여 어느 페이지든
    간편하게 적용하여 사용하실 수 있습니다.

    편집태그는 사용자태그와 사용개념은 비슷하지만 영역별로 기본 제공되는 소스로
    수정이 불가능하나 사용자태그의 경우 원하는 소스를 가져와 사용자가 자유롭게
    수정    하여 사용하실 수 있다는 차이점이 있습니다.

    사용자태그 사용 도움말

    1. 사용자태그를 사용할 기본 페이지 타입을 선택합니다.

    2. 사용자태그 생성시 입력한 태그명이 표시됩니다.

    3. 사용자태그 생성시 입력한 태그설명이 표시됩니다.

    4. 편집한 사용자태그를 현재 페이지 타입 태그목록에만 노출할 것인지

    모든 페이지 타입의 태그목록에 노출할 것인지 설정합니다.
    모든 페이지 타입으로 설정하여 사용할 경우 사용자태그에 입력한 가상태그가
    페이지 타입에 지원되지 않는 가상태그일 경우 사용이 불가능합니다.

    5. 사용자태그의 편리한 검색을 위해 사용자가 검색 키워드를 입력하실 수

    있습니다. 키워드의 구분은 쉼표(,)로 구분하여 입력해주세요.

    6. 클릭 시 태그목록 화면으로 이동합니다.

    7. 클릭 시 현재 저장되어있는“모든 페이지, 편집태그, 사용자태그”소스를

    선택하여 가져올 수 있습니다.

    8. 1번에서 선택한 페이지 타입에 따라 지원되는 태그목록이 노출됩니다.

    주의!
    - 사용자태그 편집 시 include, script, block태그는 사용하실 수 없습니다.
    - block 태그는 param 형태로 사용하실 수 없습니다. ex) block_[#P1]
